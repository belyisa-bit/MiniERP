# MiniERP

Sistema ERP simples desenvolvido para integração entre estoque, vendas e financeiro.

## 🚀 Tecnologias
- Node.js
- Express
- SQLite
- HTML
- CSS
- JavaScript

## 📦 Funcionalidades
- Cadastro de produtos
- Controle de estoque
- Registro de vendas
- Integração financeira

## ▶️ Como executar

```bash
cd backend
npm install
npm run dev
```
